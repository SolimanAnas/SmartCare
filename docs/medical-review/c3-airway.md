## 🟢 Confirmed sound
- Anatomy: tongue = #1 obstruction, gas exchange at alveoli, cricoid = narrowest in young children.
- Assessment: gurgling → suction, stridor = upper-airway, restlessness = earliest hypoxia.
- **FBAO (3.3):** mild (encourage cough) vs severe; infant **5 back blows + 5 chest thrusts** (no
  abdominal thrusts); responsive adult abdominal thrusts; unresponsive → CPR; no blind finger sweeps.
- Adjuncts: OPA only if no gag; NPA contraindicated in basal skull fracture; remove OPA if gags.
- Oxygen: target 94–98% (COPD 88–92%), NRB 10–15 L/min — correct.
- Ventilation: ~1 breath/6 s (≈10/min) adult, two-rescuer BVM for seal, gastric distension from
  over-ventilation — correct.
- Suction: on withdrawal only, ≤15 s adult, hypoxia/bradycardia risks — correct.
- Advanced airway: waveform capnography = gold standard, supraglottic above cords, DOPE — correct.
- Respiratory failure: declining mental status/rate → assist with BVM; silent chest = decompensation.

No changes required.
