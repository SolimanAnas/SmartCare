## 🔴 Error (fix before release)
1. **8.6 Q5 — placard / UN-number facts are wrong (two errors in the correct answer):**
   - **"Yellow placards are Class 4 (Flammable solids/reactive)."** ❌ A **yellow** placard is
     **Class 5 — Oxidizer (5.1) / Organic peroxide (5.2)**. Class 4 (flammable solid / dangerous-when-wet)
     is red-and-white-striped / blue.
   - **"UN 1971 is not a valid UN number."** ❌ **UN 1971 = Methane (compressed) / Natural gas**, a
     valid Class 2.1 flammable gas.
   *Fix:* rebuild this item with a correct color→class mapping, or pick a real UN number that matches
   the intended hazard. As written it teaches incorrect hazmat identification.

## 🟢 Confirmed sound (highlights)
- **Heat:** exertional heat stroke (can still sweat) vs classic; **cold-water immersion = gold
  standard**; stop active cooling ~102°F to avoid overshoot — correct.
- **Cold:** profound bradycardia with a pulse → **no compressions, gentle handling**; defibrillation
  ineffective <86°F (30°C); rewarm core before periphery; ECMO referral — correct.
- **Drowning:** hypoxic arrest → ventilation-focused, no abdominal thrusts, transport all (secondary
  drowning), diving reflex in cold-water submersion — correct.
- **Diving/altitude:** **AGE** (left lateral, hyperbaric), AMS, Type I DCS, **HACE → immediate descent +
  dexamethasone** — correct.
- **Bites/stings:** pit-viper (immobilize at/below heart, 25% dry bites), **coral snake neurotoxin**,
  stingray hot-water immersion — correct.
- **Hazmat (otherwise):** chlorine (UN 1005) upwind/uphill 500 ft; organophosphate → atropine/2-PAM;
  **cyanide → hydroxocobalamin**; irradiation vs contamination — correct.
- **CBRNE:** Sarin nerve agent, blast-scene staging for secondary devices, **smallpox vs chickenpox
  (centrifugal/synchronous vs centripetal/asynchronous)** — correct.
- **MCI/ICS:** first-unit command/declare MCI, START & JumpSTART categories, Unified Command,
  Transport Unit Leader, staging — correct.
