## 🟠 Verify / update (outdated guideline)
1. **7.1 Q4 — meconium-stained, non-vigorous newborn → "suction the mouth and trachea before
   stimulating."** This reflects **pre-2015 NRP**. **Current NRP no longer recommends routine
   tracheal (endotracheal) suctioning** for a non-vigorous infant with meconium — the priority is to
   **begin PPV** within the first minute (oral/nasal bulb suctioning only as needed for obstruction).
   *Fix:* change the answer to "open airway, clear the mouth/nose as needed, and **begin PPV** — routine
   tracheal suctioning is no longer recommended."
